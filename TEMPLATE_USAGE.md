# Prompt Template Feature

This release introduces a simple prompt template system.

- Click the `テンプレ` button next to the send button to open the template list.
- Templates can be created, deleted and applied from this modal.
- Selecting `採用` will copy the template text into the prompt box and focus it for editing.
- Templates are stored per user on the server.
